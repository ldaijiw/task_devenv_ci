```bash
git remote show origin

* remote origin
  Fetch URL: https://github.com/user/repo.git

  HEAD branch: main
  Remote branch:
    main tracked
  Local branch configured for 'git pull':
    main merges with remote main
  Local ref configured for 'git push':
    main pushes to main (up to date)
```
